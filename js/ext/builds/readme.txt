JavaScript files in this folder was created using ExtJs builder tool at
(http://extjs.com/download/build).

* formeditor.js
  Components: Button Widget, Data-Core, Data-JSON Support, Form-BaseFields,
  Form-Floating Editor

* message-box.js
  Components: Core - Utilities, Core - Layers, Core - Drag and Drop, Core - State Management,
  Button Widget, Dialog - Basic Widget(with Dialog Drag and Drop Support), Dialog - MessageBox

* structure-tree.js
  Components: Core - Utilities, Core - Drag and Drop(Overflow Scrolling Support), Core - State Management,
  Tree - Core(Drag and Drop Support), Tree - Ajax Loading Support
